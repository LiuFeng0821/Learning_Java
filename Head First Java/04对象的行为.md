# 4 对象的行为

## 4.1 方法参数与返回类型
**方法会运用形参，调用的一方会传入实参**
```java
Dog d = new Dog();
d.bark(3);  //此处的3就是实参

void bark(int numOfBarks){  //此处的numOfBarks就是形参
    while (numOfBarks > 0){
        System.out.println("ruff");
        numOfBarks --;
    }
}
```
- **值传递**：当我们调用函数时，在**形参对应的是实参的副本**。
- **引用传递**：在**函数内部看到的形参是实参本身**。我们可以认为**引用传递，形参和实参是同一个变量**。常用实现的方法是函数传递实参的地址，编译器自动帮你做由地址到值的转换。
